# Seoul D-Hub — 首爾行程 2025

家族三人共用的首爾行程討論頁面。

- **日期**：2025/7/1（二）– 7/5（六）
- **飯店**：明洞斯坦福飯店（Stanford Hotel Myeongdong）
- **成員**：Mo、老婆、女兒（13歲）

## 功能

- 五天垂直時間軸，每個地點一張卡片
- 點擊「地圖」直接開 Google Maps
- ＋新增地點：任何人都能加入想去的地方，標記是誰提議的
- 右側總覽列：全程時間軸 + 票價一覽
- 上方機票資訊欄（班號、時間）
- 每天底部備註欄，自動儲存（localStorage）
- 備份按鈕：匯出 JSON，方便統整討論紀錄
- 列印 / 存 PDF 按鈕
- 鍵盤 ← → 切換日期

## 部署

GitHub Pages 靜態網站，單一 HTML 檔案，無需任何 build 工具。
